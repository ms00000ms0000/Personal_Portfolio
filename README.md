# 📖 Overview

This project is a **next-generation Personal Portfolio** designed to present technical expertise through a clean, interactive and visually engaging experience.

Unlike traditional portfolio websites, this application combines modern frontend technologies with immersive animations to create a professional developer portfolio suitable for recruiters, hiring managers and technical interviews.

The website highlights:

- Professional introduction
- Technical skills
- Experience
- AI & Machine Learning projects
- Certifications
- Achievements
- Contact information
- Responsive design
- Smooth animations
- Interactive 3D graphics

The entire project follows a modular architecture, making it easy to maintain, extend and scale.

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/ms00000ms0000/Personal_Portfolio.git
```

Move inside the project

```bash
cd Personal_Portfolio
```

Install dependencies

```bash
npm install
```

---

# ▶ Running Locally

Start the development server

```bash
npm run dev
```

Open

```text
http://localhost:5173
```

---

# 📦 Production Build

Generate an optimized production build

```bash
npm run build
```

Preview production build

```bash
npm run preview
```

Production URL

# 🛠 Customization Guide

Updating the portfolio is straightforward.

### Personal Information

Edit:

```text
src/constants/data.js
```

Update:

- Name
- Profile Image
- Resume
- Social Links
- Email
- Roles

---

### Skills

Modify:

```javascript
skills: [
  ...
]
```

---

### Projects

Add or edit:

```javascript
projects: [
  {
    title: "",
    description: "",
    github: ""
  }
]
```

---

### Experience

Update:

```javascript
experiences: [
  ...
]
```

---

### Certifications

Edit:

```javascript
certifications: [
  ...
]
```

---

### Achievements

Update:

```javascript
achievements: [
  ...
]
```

---

After making changes simply run

```bash
git add .

git commit -m "Updated portfolio"

git push origin main
```

Netlify will automatically redeploy the latest version.
